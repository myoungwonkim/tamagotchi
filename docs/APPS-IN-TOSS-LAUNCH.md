# 앱인토스 출시·광고 QA 체크리스트

## 빌드·환경

- [ ] `npm install` 성공
- [ ] `npm run build:ait` → `dist/` 생성, `.ait` 번들 생성
- [ ] `granite.config.ts` appName = `abysspet`, displayName / icon = 콘솔과 일치
- [ ] 첫 화면 **10초 이내** (저사양 기기)
- [ ] GitHub Pages 정적 빌드 회귀 (`python3 -m http.server` + `index.html`)

## 샌드박스 광고 (테스트 ID)

- [ ] `ait-ad-test-interstitial-id` — T1 게임오버 후 전면형
- [ ] `ait-ad-test-rewarded-id` — R1 부활 보상형
- [ ] load → show → dismissed → reload 패턴
- [ ] 광고 중 게임 소리 멈춤, 종료 후 재개
- [ ] 광고 로드 실패 시 게임 정상 진행

## 게임 플로우 + 광고

| # | 시나리오 | 기대 |
|---|----------|------|
| L1 | 방치 사망 → «새 펫» | T1 전면형 후 이름 모달 |
| L2 | 사망 → «광고 보고 부활» | R1 시청 후 펫 복원, 게임오버 닫힘 |
| L3 | 부활 후 같은 사망에서 재부활 | 버튼 비활성 또는 숨김 |
| L4 | 성체 진화 | T2 전면형 1회 (세션) |
| L5 | 졸업 → 새 알 | T3 전면형 |
| L6 | 30분+ 오프라인 복귀 | T4 전면형 1회 (세션) |
| L7 | 8분 내 연속 게임오버 | 전면형 2회째 스킵 (쿨다운) |
| L8 | 세션 전면형 4회 시도 | 3회까지만 노출 |
| L9 | 스탯 위험 → 응급 돌봄 | R2 보상형 |
| L10 | 방치 임박 → 타이머 리셋 | R3 보상형 |
