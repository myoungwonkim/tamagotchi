탐색정 — 레이아웃 재구성 시안

현재 게임 UI 구조(헤더·펫·스탯·2×2 버튼)를 바꾼 6종입니다. 수중 탐색정 컨셉 유지. 게임 미반영.

구조 변경 탐색정·잠수함 모바일 세로 기준
L1 · HUD

함교 오버레이

스탯 → 모서리 미니 게이지 · 펫 → 중앙 원형 · 버튼 → 하단 1×4 레버

FPS/비행 시뮬 함교. 헤더 없이 HUD만. 정보 밀도 높음.

  • 펫 이름·수심은 상단 좌우 코너
  • 말풍선은 관측창 내부
L2 · SPLIT

좌우 콘솔

좌: 뷰포트+말풍선 · 우: 스탯 세로 + 2×2 버튼

실제 잠수정 조종석 — 화면과 계기가 분리된 배치.

  • 한 손 엄지로 오른쪽 조작에 최적화
  • 가로 모드에서도 자연스러움
L3 · IMMERSIVE

전면 뷰포트

펫 75% · HUD 오버레이 · 스탯 1×4 스트립 · 아이콘 독

관측에 몰입. 스탯·버튼은 최소 UI로 하단에만.

  • 성체 펫 크기 극대화에 유리
  • 도감·음소거는 오버레이 버튼으로 처리 가정
L4 · DIAL

포르톨 시계

스탯 → N/S/E/W 4방 · 펫 → 중앙 · 버튼 → 원형 4개

압력 게이지판처럼 펫 주위에 수치가 둘러싸는 배치.

  • 스탯 4개만 둥글게 — 공간 효율
  • 버튼이 둥글어 ‘밸브’ 연상
L5 · LOG

잠수 일지

세로 스크롤 · 표본 스케치 · 인용 · 스탯 줄글 · 2×2 버튼

탐사 보고서 UI. 펫은 ‘표본 도면’ 블록, 대사는 인용문.

  • 현재 UI와 가장 다른 정보 구조
  • 도감·대사 톤과 세계관 통합 용이
L6 · SCOPE

잠망경·어뢰관

타원 스코프 · 하단 4 튜브(스탯) · 1×4 키패드

군용 잠수함 잠망경 + 어뢰 발사管 메타포. 십자선 오버레이.

  • 기괴·긴장감 연출에 특화
  • 스탯이 ‘管’ 형태로 시각화
