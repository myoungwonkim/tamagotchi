# 게임물 등급 심의 체크리스트 (GRAC)

앱인토스 **게임 미니앱** 출시 전 필수입니다. ([오픈 프로세스](https://developers-apps-in-toss.toss.im/intro/onboarding-process.html))

## 서비스 개요 (심의 제출용)

| 항목 | 내용 |
|------|------|
| 게임명 (한글) | 어비스펫: 심해 가상 펫 |
| 게임명 (영문) | Abyss Pet |
| 부제 | 심해 가상펫 육성 및 수집 게임 |
| 장르 | 시뮬레이션 / 육성 |
| 플랫폼 | 앱인토스 WebView 미니앱 (`appName`: `abysspet`) |
| 이용 대상 | 전체 이용가 예상 (폭력·선정성 없음) |
| 결제 | 인앱 광고(전면형·보상형), 유료 결제 없음(1차 출시) |
| 개인정보 | 토스 로그인 미사용, 펫·도감 데이터는 기기 localStorage |
| 고객문의 | nolsoop.games@gmail.com |

## 심의 전 준비물

- [ ] 사업자등록증 사본
- [ ] 게임 설명서 (플레이 방법, 사망·부활 규칙, 광고 노출 시점) → [GRAC-SUBMISSION-PACK.md](./GRAC-SUBMISSION-PACK.md)
- [ ] 스크린샷 5장 이상 (메인, 돌보기, 진화, 게임오버, 탐사 일지)
- [ ] 시연 영상 또는 빌드 (샌드박스 `.ait` / 검수용 QR)
- [ ] 등급분류 신청서 ([게임물관리위원회](https://www.grac.or.kr))

## 게임 내 콘텐츠 자가 점검

- [ ] 폭력·혈흔·살해 묘사 없음
- [ ] 선정성·도박·사행성 없음
- [ ] 욕설·혐오 표현 없음 (성체 대사 tier 포함 전수 확인)
- [ ] 사망 표현: `heart-broken` 스프라이트 + 이별 메시지 (잔혹 묘사 없음)
- [ ] 보상형 광고: 사용자 **자발적** 선택 (부활·응급돌봄 버튼)
- [ ] 전면형 광고: 게임오버·마일스톤 등 **전환 구간**만 (돌보기 중 미노출)

## 제출 후

- [ ] 등급분류증명서 PDF 확보
- [ ] 앱인토스 콘솔 검수 요청 시 증명서 첨부
- [ ] 등급 표시 문구·아이콘을 게임 내 또는 스토어 설명에 반영 (심의 결과에 따름)

## 대안 경로 (자체등급분류사업자)

앱스토어·플레이스토어 등에 **선출시** 후 링크를 앱인토스에 제출하는 방식. 별도 앱 패키징이 필요하므로 GRAC 직접 심의를 권장합니다.

## 예상 일정

|------|------|

| 앱인토스 검수(병렬 가능) | 2~3영업일 |
