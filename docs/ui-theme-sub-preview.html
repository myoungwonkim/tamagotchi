수중 탐색정 UI 시안

현재 게임 레이아웃·버튼 배치를 유지한 채, 잠수정·탐사함·관측구 느낌만 바꿨습니다. 게임 미반영.

현재 구조 유지 관측창·수압·계기판 픽셀 스프라이트 호환
기준

현재

크림·핑크 모바일 UI. 탐색정 느낌 없음 — 비교용.

탐색정

SUB-1 — 황동 구체

1930년대 황동 잠수구. 둥근 관측창, 리벳, 안은 따뜻한 금속·밖은 푸른 수중.

  • 현재 UI의 둥근 카드 → 황동 링 관측창으로 치환
  • 말풍선은 창 안쪽 하단에 배치
탐색정

SUB-2 — 알빈 함교

현대 심해 연구잠수정. 밝은 회색 패널, 직각, 수심·수압 디스플레이.

  • 현재 버전과 가장 가까운 ‘현실적’ 탐색정
  • 가독성 최우선 — 야간·수중 촬영 다큐 톤
추천

SUB-3 — 수압 계기판

현재 레이아웃을 심해함 내부로. 청회색 선체, 스탯은 수압 게이지, 숫자는 계기값.

  • 현재 구조 그대로 + 색·질감만 탐색정화
  • 게임 적용 난이도·효과 밸런스 최적
탐색정

SUB-4 — 소나르 돔

소나어 돔 안에서 관측. 녹색 인광, 원형 스캔, 함교 터미널 느낌.

  • 음침함·기괴 펫과 궁합
  • 모노스페이스 UI — 계기판 연상
탐색정

SUB-5 — 낡은 탐사정

오래된 국가 탐사 잠수정. 녹슨 선체, 비상 수심계 빨강, 유막 낀 관측창.

  • 스탯 패널에 HULL INTEGRITY 라벨
  • 불량 심해어·송곳니어 세계관과 맞음
1순위: SUB-3 수압 계기판 — 현재 UI에서 색·타이포·게이지만 바꿔도 탐색정 느낌이 납니다. 관측창 연출을 강조하려면 SUB-1, 현실 연구정 느낌은 SUB-2, 분위기 극대화는 SUB-4·5입니다.
