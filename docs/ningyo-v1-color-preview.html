변형 A(청록 기본)를 기준으로 색을 다양화하고 얼굴을 약간 덜 험상궂게 조정한 3종 — 스테이징 전용(게임 미반영)

기준 · 변형 A (청록 기본)

A-1 · 청록 + 골드

청록 비늘에 은은한 앤티크 골드 테두리와 금빛 지느러미. 따뜻하고 고급스러운 톤.

원본 방향
반전(우측)

청록 몸에 산호빛 지느러미와 발그레한 볼. 부드럽고 다정한 톤.

원본 방향
반전(우측)
