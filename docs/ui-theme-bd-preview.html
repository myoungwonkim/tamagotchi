B × D 확장 시안

페미콤 심해(B)의 창·테두리·스캔라인과, 염수 침식(D)의 침수 금속·계측기·앰버 포인트를 섞은 5종입니다. 게임 미반영.

B — 페미콤 심해크림 프레임, 16색, 스캔라인, 세그먼트 스탯, 고딕·고대비
D — 염수 침식무채색 침수, 부유물, 모노스페이스, 낡은 전구색, 얇은 막대
B+D

BD-1 — 침수 세가 창

B 3px 창 골격 · D 침식 크림·먼지·얇은 막대

페미콤 대화창 형태는 유지하되, 테두리를 녹슨 크림으로 바꾸고 안쪽은 D처럼 침묵.

  • 가장 직관적인 B+D 50:50
  • 스탯 색도 채도 낮춤 — 네온 느낌 제거
B+D

BD-2 — 계측 프레임

D 침수 패널·먼지 · B 행마다 크림 프레임·원색 막대

전체는 D 계측기. 스탯 한 줄마다만 페미콤 선택지 박스를 씌움.

  • 정보 UI는 D, 강조만 B 원색
  • 펫 영역은 프레임 없이 심해 공허
B+D

BD-3 — 염수 CRT

B 스캔라인·세그먼트·그림자 · D 앰버 인광·침식 테두리

낡은 CRT 모니터가 염수에 잠긴 느낌. 텍스트는 앰버, 막대만 B 원색.

  • 레트로 감성이 가장 강함
  • 장시간 플레이 시 눈 피로 주의 — 옵션으로 스캔라인 끄기 고려
B+D

BD-4 — 침수 아케이드

B 두꺼운 베젤·창 UI · D 금속·먼지·채도 죽은 버튼

심해에 가라앉은 아케이드 기계. 페미콤 베젤이 두껍고 표면은 D 금속 질감.

  • ‘오락실+공포’ 톤 — 기괴 펫과 궁합 좋음
  • 버튼 아이콘 opacity 낮춰 낡음 표현
추천

BD-5 — 관측 로그

B 펫 창 프레임·이름 그림자 · D 로그 본문·얇은 막대·앰버

잠수정에서 심해 생물을 기록하는 터미널. B는 ‘관측창’에만, 나머지는 D.

  • B+D 중 게임 적용 밸런스 최적
  • 스탯 라벨 › 접두 — 로그 느낌
  • 어종만 시안(cyan) 포인트 — B 색 최소 사용
1순위: BD-5 관측 로그 — B의 레트로는 펫 창에만, D의 음침함은 전체 톤에. 더 레트로하게 가려면 BD-3, 베젤·아케이드 느낌은 BD-4, 가장 무난한 혼합은 BD-1입니다.
