스프라이트는 단일 PNG라 창·방패는 몸과 함께 움직입니다. 실제 적용 시 능어처럼 컨테이너 이동 + 스프라이트 반전에 전진(lunge)·기울임(rotate)을 더해 공격/방어 느낌을 냅니다.
A · 기본

순찰 (능어와 동일)

좌우 왕복 + 오른쪽 이동 시 반전. 호위병이 구역을 도는 느낌.

B

창 찌르기 (제자리)

제자리에서 좌·우로 번갈아 창을 내미는 느낌. 이동 없이 idle 대용.

C

방패 태세

좁게 좌우로 흔들리며 방패를 든 채 경계. 공격보다 수비·대기 무드.

D · 추천

순찰 + 창 찌르기

왕복 이동 중 양 끝에서 잠깐 멈춰 창을 찌름. 호위+경계가 한 번에.
