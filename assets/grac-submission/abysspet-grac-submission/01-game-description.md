## 1. 게임 개요

«어비스펫»은 심해를 배경으로 한 **무료 가상 펫 육성·수집 게임**입니다.  
플레이어는 불가사의한 알을 돌보며 5단계(알 → 라바 → 치어 → 청소년어 → 성체)로 진화시키고,  
성체 펫은 «비밀 탐사 일지»(도감)에 기록합니다.

- **대상 연령:** 전체 이용가 예상 (비현실적 상어 위협 연출은 있으나 혈흔·상해 묘사 없음)
- **플레이 시간:** 세션 단위 자유 (짧은 휴식형 육성)
- **멀티플레이:** 없음
- **소셜·채팅:** 없음
- **계정·로그인:** 없음 (토스 로그인 미사용)

## 2. 플레이 방법

### 2.1 기본 돌보기

메인 화면에서 다음 4가지 행동으로 펫의 스탯을 관리합니다.

| 행동 | 효과 |
|------|------|
| 먹이 주기 | 배고픔 증가 |
| 놀아주기 | 행복 증가 |
| 씻기기 | 청결 증가 |
| 재우기 / 깨우기 | 수면 상태 전환 (잠든 동안 스탯 감소 완화) |

스탯: **배고픔·행복·청결·건강** (0~100). 평균 케어가 낮으면 건강이 서서히 감소합니다.

### 2.2 진화

| 단계 | 조건 (실시간 누적) |
|------|-------------------|
| 알 | 시작 |
| 라바 | 약 0.36일 |
| 치어 | 약 1.07일 |
| 청소년어 | 약 2.5일 |
| 성체 | **5일** (알→성체 총합) |

성체 진화 시 «예쁜»·«평범»·«괴기» 등 **케어 평균에 따른 외형 변형**이 결정됩니다.  
현재 출시 버전은 **심해어·심해인어 2종 테마** 중 하나로 시작합니다. 열수구 테마는 도감에서 «업데이트 예정»으로 표시되며 플레이할 수 없습니다.

### 2.3 탐사 일지 (도감)

성체가 된 펫은 탐사 일지에 자동 등록됩니다.  
심해어·심해인어 탭은 각각 9칸으로 구성되며, 현재 각 테마에서 6종의 성체를 수집할 수 있습니다. 미수집 성체는 봉인 도장으로 가려지고, 미출시 슬롯은 «업데이트 예정»으로 표시됩니다. (수집 요소, 전투·경쟁 없음)

### 2.4 졸업 (새 알)

성체 펫을 키운 뒤 «새 펫 시작»을 선택하면 현재 펫은 도감에 남고, 새 알부터 다시 육성합니다.

## 3. 사망·게임 오버

펫은 다음 경우 **사망(이별)** 합니다. 잔혹·혈흔·신체 훼손 묘사는 없습니다. 건강·방치 사망은 하트 깨짐 아이콘과 이별 메시지로, 상어 습격은 짧은 접근·화면 플래시 후 유령 캐릭터로 표현됩니다.

| 원인 | 조건 |
|------|------|
| 건강 붕괴 | 건강 0 이하 |
| 방치 | 평균 케어 10 미만 상태가 **10분** 지속 |
| 상어 습격 | 알 단계를 제외한 활성 플레이 중 낮은 확률로 발생 (출생·부활 직후 3분 및 수면 중에는 발생하지 않음) |

게임 오버 후 선택지:

1. **새 펫 키우기** — 새 이름 입력 후 처음부터 시작
2. **광고 보고 부활** (보상형, 선택) — 사망 직전 상태로 1회 복원

## 4. 광고 (수익 모델)

**유료 결제 없음.** 인앱 광고만 사용합니다.

### 4.1 전면형 (인터스티셜)

| 시점 | 설명 |
|------|------|
| 게임 오버 → 새 펫 | 전환 구간 |
| 성체 진화 알림 후 | 마일스톤 (세션 1회) |
| 졸업 확인 후 | 전환 구간 |
| 30분+ 오프라인 복귀 | 복귀 알림 후 (세션 1회) |

**미노출:** 돌보기 버튼 직후, 진화 연출 중, 첫 10분 튜토리얼 구간.  
세션당 최대 3회, 최소 간격 8분.

### 4.2 보상형 (리워드)

| 보상 | 조건 | 사용자 선택 |
|------|------|-------------|
| 부활 | 게임 오버 | «광고 보고 부활» 버튼 (사망당 1회) |
| 응급 돌봄 | 스탯 위험 | 선택 버튼 |
| 방치 타이머 리셋 | 케어 매우 낮음 | 선택 버튼 |

보상은 광고 시청 **완료 후**에만 지급됩니다.
